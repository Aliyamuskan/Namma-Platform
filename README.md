# Namma-Platform 🚆

Namma-Platform is an Android railway assistance app designed for small railway stations and rural passengers. The app provides platform information, train details, visual coach layouts, and Kannada voice announcements to help users easily identify where to board trains.

## Features

* Live train and platform updates
* Coach layout visualization
* Kannada Text-To-Speech announcements
* Multilingual support
* User-friendly high-contrast UI

## Technologies Used

* Kotlin
* XML
* Firebase Realtime Database
* JSON & Gson
* RecyclerView
* Android Text-To-Speech (TTS)

## Purpose

To improve accessibility and reduce confusion for rural, elderly, and non-English-speaking railway passengers.
